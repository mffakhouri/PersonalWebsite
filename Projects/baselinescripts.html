Baselining Scripts

github.com/mikefak/CySec-ScriptsNPrograms/

This repo will hold a collection of security scripts that some fellow peers and I are actively contributing to. These scripts are made with the intention of learning the fundamentals of bash scripting and the proper baselining of linux systems.

usermng.py - A python script that is able to take a list of authorized users (entered either manually or through a text document) and list discrepancies in comparison to the /etc/passwd file.

AIF.sh - A shell script that I contributed to YES (Yorge's Enumeration Script) that establishes basic system information inspired by the Security Daemon's Asset Inventory inject. Such information includes system name, O/S and version, intenal/external addresses, accounts, services running, and open ports.

PassPol.sh - A shell script that simplifies the addition of password policies utilizing libpam-pwquality. This includes complexity requirements, length minimum, and failed attempt penalities.

mysqlbackup.sh - A shell script that performs a full backup of current MySQL databases and saves them to a desired directory.
