Firewall-based Scripts

github.com/mikefak/FWscripts

These scripts are made with the intention of learning adequate host-based security along with their applications in competitive scenarios. Most will be in the realm of the iptables utility.

IPTS.sh - A shell program that applies iptables rules for incoming/outgoing connections in accordance to open ports on the active machine. It also simplifies some features of iptables, such as saving/loading rules, listing rules, establishing logging, and accepting/dropping all incoming packets.

WorkstationHostFW.sh - A shell script used in a practice security competition to establish basic host-based firewall rules.
