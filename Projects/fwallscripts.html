Firewall-based Scripts

github.com/mikefak/FWscripts

These scripts are made with the intention of learning adequate host-based security along with its applications in a competitive format. Most will be in the realm of the iptables utility.


IPTS.sh - A shell program that applies iptables rules for incoming/outgoing connections in accordance to open ports on the active machine, along with other preset applications. It also simplifies some features of iptables, such as saving/loading rules, listing rules, establishing logging, and accepting/dropping all incoming packets.

WorkstationHostFW.sh - A shell script that serves as an example for a possible host firewall, covering ntp, internet access, ldap authorization, and logging.
